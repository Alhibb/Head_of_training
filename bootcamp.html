KIDS HOLIDAY BOOTCAMP PROGRAM

This is the proposed course curriculum for the Kids Holiday BootCamp for 6 weeks. The program shall run for 6 weeks, covering different emerging technology courses each week. The curriculum ensures that students get hands-on experience with each technology, work on real projects, and present their work, enhancing their learning and communication skills.

Aims:

  • To introduce children to emerging technologies and inspire interest in STEM (Science, Technology, Engineering, and Mathematics) fields.
  • To provide hands-on learning experiences that promote creativity, critical thinking, and problem-solving skills.
  • To encourage collaborative learning and teamwork among students.

Objectives:

  • By the end of the program, students will have a foundational understanding of coding, robotics, AI, 3D modeling, cybersecurity, and game development.
  • Students will complete practical projects that demonstrate their learning and ability to apply new skills.
  • Students will develop confidence in their abilities to tackle technological challenges and present their work to others.

Presented by Ibrahim Rabiu, Head of Training at BT Hub

Week 1: Introduction to Scratch

Goals

Learn the basics of Scratch, animations, and game design principles.

KPIs

  • Create an animated greeting card
  • Develop a short interactive story with dialogues
  • Create an animated sprite dance party
  • Design a basic clicker game with increasing difficulty
  • Collaborative project: Mini platformer game

Projects

Create a mini platformer game combining learned concepts.

Activities

  • Creating a simple animation
  • Building a basic interactive story
  • Designing and animating characters
  • Creating a simple game
Week 2: Introduction to Robotics

Goals

Understand the basics of robotics using Lego Mindstorms kits and programming.

KPIs

  • Build a basic moving robot
  • Program the robot to follow a set path
  • Create a robot that stops when it detects an obstacle
  • Design and program a robot to navigate an obstacle course
  • Collaborative project: Robot to complete a specific task

Projects

Build and program a robot to complete a specific task.

Activities

  • Building a simple robot
  • Programming the robot to move and turn
  • Integrating sensors
  • Designing and programming solutions for challenges
Week 3: Introduction to Web Design

Goals

Learn basic HTML5, CSS, and responsive web design principles.

KPIs

  • Create a simple webpage with a title, headings, and paragraphs
  • Create a webpage with semantic elements
  • Style a webpage with colors, fonts, and layouts
  • Create a responsive webpage that adjusts to different screen sizes
  • Collaborative project: Personal homepage or small website

Projects

Create a personal homepage or a small website combining learned concepts.

Activities

  • Explore basic HTML5 and CSS
  • Use semantic elements in HTML5
  • Apply styles to HTML elements
  • Learn about media queries and responsive design principles
Week 4: Basic Computer Skills

Goals

Understand basic computer operation, file management, and software applications.

KPIs

  • Identify and label computer components
  • Organize files and folders on a computer
  • Create a document, spreadsheet, and presentation
  • Perform web searches and practice safe browsing habits
  • Collaborative project: Small project using various computer skills

Projects

Create a small project using various computer skills.

Activities

  • Understanding the basics of computer operation
  • Managing files and folders
  • Using word processing, spreadsheet, and presentation software
  • Browsing the web safely and effectively
Week 5: Introduction to Cybersecurity

Goals

Learn about cybersecurity concepts and online safety practices.

KPIs

  • Understand the importance of strong passwords
  • Identify common cyber threats
  • Practice safe online behavior
  • Understand the basics of data encryption
  • Collaborative project: Create a cybersecurity awareness poster

Projects

Create a cybersecurity awareness poster.

Activities

  • Learn about strong passwords and password managers
  • Identify phishing attempts and other online scams
  • Practice safe browsing and online communication
  • Explore basic data encryption techniques
Week 6: Microsoft PowerPoint Basics

Goals

Learn the basics of creating and designing PowerPoint presentations.

KPIs

  • Create a simple presentation with text and images
  • Design visually appealing slides with consistent styles
  • Enhance a presentation with multimedia elements
  • Create an engaging presentation with animations and transitions
  • Collaborative project: Presentation on a topic of choice

Projects

Create a presentation on a topic of choice combining learned concepts.

Activities

  • Explore basic features of Microsoft PowerPoint
  • Use themes, layouts, and design templates
  • Add and edit images, videos, and audio
  • Animate text and objects and apply slide transitions
