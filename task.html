JSCE Bootcamp - HTML Task

Task Instructions:

You have learned about the basic building blocks of HTML. Now it's time to put your knowledge into practice! Create an HTML page based on the following guidelines:

Page Structure:

  • Use the correct doctype declaration.
  • Include the essential html, head, and body elements.
  • Give your page a relevant title using the title tag (e.g., "My First Website").

Content:

  1. Heading: Start with a main heading (h1) that describes your page's topic.
  2. Paragraphs: Write at least three paragraphs (p) with some information related to the heading.
    • Use bold (b or strong) and italic (i or em) formatting to emphasize certain words or phrases.
  3. Lists: Create one unordered list (ul) and one ordered list (ol).
    • Each list should have at least three items (li).
  4. Marquee: Add a scrolling text using the marquee tag.
    • Experiment with different directions (direction="left/right/up/down") and behaviors (behavior="scroll/slide/alternate").
  5. Table: Create a simple table (table) with two columns and three rows.
    • Use table headings (th) for the column names.
    • Add some data to the table cells (td).
  6. Images: Add an image to your page using the img tag.
    • Use the alt attribute to provide a description of the image.
  7. Audio: Embed an audio file using the audio tag.
    • Provide controls for play/pause.
  8. Video: Embed a video using the video tag.
    • Provide controls for play/pause and volume.
