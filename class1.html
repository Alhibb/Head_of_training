HTML Tags

Basic Structure

These tags define the fundamental structure of an HTML document.

<!DOCTYPE html>

Definition: Declares the document as HTML5, instructing browsers to render it using the latest HTML version.

Example: <!DOCTYPE html>

<html>

Definition: The root element of every HTML page. All other elements are nested within it.

Example:

<html>
  <head> ... </head>
  <body> ... </body>
</html>
<head>

Definition: Contains metadata about the HTML document, such as the title, links to stylesheets, scripts, and other information not displayed directly on the page.

Example:

<head>
  <title>My Website</title>
  <link rel="stylesheet" href="style.css">
</head>
<title>

Definition: Specifies the title of the HTML document, which is displayed in the browser's title bar or tab.

Example: <title>Ibrahim's Page</title>

<body>

Definition: Encloses the visible content of the HTML document, including text, images, links, and more.

Example:

<body>
  <h1>Welcome to BT_HUB</h1>
  <p>This is the main content of the page.</p>
</body>

Headings (<h1> to <h6>)

Use headings to structure content with different levels of importance.

<h1> to <h6>

Definition: Headings define six levels of importance, with <h1> being the most important and <h6> the least.

Examples:

<h1>Welcome to BT_HUB</h1>
<h2>About Maryam</h2>
<h3>My Skills</h3>

Paragraphs (<p>)

<p>

Definition: Represents a paragraph of text.

Examples:

<p>Abba is visiting BT_HUB today.</p>
<p>He is interested in learning about their latest technology.</p>

Text Formatting

<strong>

Definition: Makes text bold, indicating strong importance.

Example: <p>This is a <strong>very important</strong> message.</p>

Output: This is a very important message.

<em>

Definition: Makes text italic, often used for emphasis or stylistic purposes.

Example: <p>Ibrahim whispered, <em>Let's meet at BT_HUB</em>.</p>

Output: Ibrahim whispered, Let's meet at BT_HUB.

Horizontal Rule (<hr>)

<hr>

Definition: Inserts a horizontal rule, visually separating content sections.

Example: <p>First section</p><hr><p>Second section</p>

Output:

First section

Second section

Lists

<ul> - Unordered List

Definition: Represents a list of items in no particular order, typically rendered as bullet points.

Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Output:

  • Item 1
  • Item 2
  • Item 3
<ol> - Ordered List

Definition: Represents a list of items in a specific order, typically rendered as numbered items.

Example:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Output:

  1. First item
  2. Second item
  3. Third item
<li> - List Item

Definition: Represents a single item in a list.

Example: <li>Item</li>

Links

<a> - Anchor

Definition: Defines a hyperlink, used to link from one page to another.

Example: <a href="https://example.com">Visit Example</a>

Output: Visit Example

Images

<img> - Image

Definition: Embeds an image in the document. Requires the src attribute to specify the image URL and the alt attribute for alternative text.

Example:

<img src="image.jpg" alt="Description of image">

Tables

<table> - Table

Definition: Represents tabular data, consisting of rows and columns.

Example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ibrahim</td>
    <td>25</td>
  </tr>
</table>

Output:

Name Age
Ibrahim 25

Forms

<form> - Form

Definition: Represents an interactive area containing form controls like input fields, checkboxes, radio buttons, and submit buttons.

Example:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <br>
  <input type="submit" value="Submit">
</form>
