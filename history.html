HTML History

HTML 1.0 (1993)

Invented by Tim Berners-Lee

Basic structure, headings, paragraphs, links, and images

HTML 2.0 (1995)

Added forms, tables, and client-side scripting

Improved structure and formatting options

Introduced HTML documents as a standard

HTML 3.0 (1995)

Tables with borders and alignment

Applets (Java-based interactive content)

Improved font control

HTML 4.0 (1997)

Stylesheets (CSS) for layout and design control

Scripting languages (JavaScript) for dynamic content

Frames for dividing content into separate sections

Improved accessibility features

HTML 4.01 (1999)

Minor updates and bug fixes

Last version of HTML before XHTML

XHTML 1.0 (2000)

Combined HTML and XML features

Stricter syntax and error handling

Replaced HTML 4.01 as the new standard

XHTML 1.1 (2001)

Modularization (reusing code modules)

Improved accessibility features

Last version of XHTML before HTML5

HTML5 (2014)

Semantic elements (header, footer, nav, etc.)

Multimedia support (video, audio, canvas)

Offline storage and caching

Improved error handling and syntax

Support for mobile devices and touch events

Currently the latest version of HTML

HTML5 Overview

HTML5 is the fifth major revision of the HTML standard

Introduced new features, improved existing ones, and deprecated others

Focuses on web application development, multimedia, and semantic structure

HTML5 Modules

Semantic Elements Module

  • Defines elements that provide meaning to the structure of a web page
  • Examples: <header>, <nav>, <article>, <section>, <aside>, <footer>

Multimedia Module

  • Defines elements for adding multimedia content to web pages
  • Examples: <video>, <audio>, <source>, <track>

Canvas and Graphics Module

  • Defines elements for creating dynamic graphics and interactive content
  • Examples: <canvas>, SVG (Scalable Vector Graphics)

Forms and Input Module

  • Defines new input types and attributes for creating more comprehensive forms
  • Examples: <input type="email">, <input type="tel">, <input type="search">, placeholder attribute, autofocus attribute, required attribute

Storage and Offline Module

  • Defines APIs for storing data locally on the client-side and enabling offline web applications
  • Examples: Local Storage, Session Storage, Offline Web Applications

Performance and Integration Module

  • Defines APIs for improving performance and integrating web applications with other systems
  • Examples: Web Workers, Web Sockets, Geolocation API, Drag and Drop API, History API

HTML5 Removed Elements

  • <font>
  • <center>
  • <strike>
  • <applet>
  • <basefont>
  • <big>
  • <blink>
  • <marquee>
